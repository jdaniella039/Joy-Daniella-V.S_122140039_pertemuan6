# Pyramid CRUD Mahasiswa API

Aplikasi backend sederhana menggunakan **Python Pyramid** dan **SQLAlchemy** untuk mengelola data mahasiswa.

---

## 🚀 Fitur

- 🔐 Dummy Autentikasi (siap dikembangkan)
- 🧾 CRUD Mahasiswa
- 📦 RESTful API
- 🧪 Unit Test Sederhana
- 🐘 Gunakan SQLite (bisa diganti PostgreSQL)
